    • HEXO上传GitHub发生错误

    今天使用hexo上传博客的时候发生了一个错误, 导致我不能将文章上传到GitHub Page.

    不知道为什么, 经过各种搜索后找到解决方法

    方法:
    把 _config.yml 文件中 repository: https://github.com/weikx/weikx.github.io.git 修改为 ssh://git@github.com:weikx/weikx.github.io.git

    这是我搜索到的相关问题, 花了好大功夫

    https://segmentfault.com/q/1010000003734223
    https://github.com/hexojs/hexo/issues/1503

    NO!!!
    经过一番操作后上面的方法并没有解决的我的问题, 并且出现了新的问题.

    由于一直卡在这里不动, 我就想知道tags/\345\256\236\347\224\250/index.html这是哪个文件
    把这个转换为汉字的方法是输入命令 git config --global core.quotepath false
    转换后输出tags/错误/index.html, 我便删除了这个目录, 从新编译上传. 后发现还是会卡在另一个文件下.

    终极解决方案
